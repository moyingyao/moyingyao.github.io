分类+位置
14年开始热门起来

R-CNN

基于区域的CNN
步骤：

  1. 随机扣一块区域，输入至CNN，看是否为想要识别的物体，选择性搜索、出框，一般2000次
  2. 将框Resize改变成同样大小的框
  3. 输入至CNN中，生成256*256的图像
  4. 使用SVM分类

Fast R-CNN

进步：将起初的先出框再卷积改为先卷积再出框，减少了卷积次数
步骤：

  1. 先卷积，再出框，一般为2000个区域
  2. ROI Pooling框出区域，值取max
  3. softmax，多类输出

RPI Pooling需要输入：

  1. 卷积后的结果
  2. 生成区域框的边框，标签为左下角和右上角的坐标

Faster R-CNN

SSD单发多框检测器

图片 → 卷积 → 出框
softmax →n+1分类（n类+背景）→ 卷积 → 出框
